Membuat Dashboard Menggunakan Python - Plotly Dash Tutorial

Feb 13, 2020 2020-02-13T15:00:00+07:00 by Malik Kurnia
Updated Mar 3 2021-03-03T21:47:38+07:00 1 min

Sekapur Sirih

Learning by doing adalah cara paling efektif dalam mempelajari sesuatu hal. Dalam artikel ini, setidaknya akan saya memberikan langkah-langkah dalam membuat dashboard visualisasi suatu data menggunakan python. Framework yang dipakai adalah Dash, dan dashboard akan dibuat dengan kontruksi sebuah opsi dropdown dan 2 reactive-graph.

Dashboard

Framework Dash dikembangkan sebagai open-source library oleh Plotly yang dibagun menggunakan Flask, Plotly.js, dan React.js. Sehingga Dash sendiri sangat cocok diimplementasikan di website untuk berbagi visualisasi insight data.

Jika kalian tertarik dengan interactive-chart dengan Python, Plotly - An Interactive Charting Library adalah tulisan yang menarik untuk dibaca. Untuk guide mengenai teknik dasar visualisasi dapat membaca Basic rules for good looking slides and dashboards.

Mempunyai skill CSS dan HTML cukup memabntu dalam pembuatan dashboard menggunakan Dash. Untuk yang belum santai saja, saya akan menjelaskan langkah-langkahnya.
